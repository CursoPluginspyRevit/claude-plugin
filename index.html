Como funciona

Da ideia ao plugin instalado

Fluxo end-to-end. Você não precisa decorar nomes de classe, lembrar de encoding ou caçar BuiltInParameter. A IA faz isso.

1

Planejar

Entrevista em 5 camadas. Escopo, gatilho, lógica, UI, edge cases.

2

Criar bundle

Pasta, script, ícone dual 96×96, layout.yaml. Tudo no padrão pyRevit.

3

Escrever código

9 regras absolutas aplicadas. APIs validadas contra dicionário.

4

Auditar

Varredura das 30 armadilhas conhecidas. Sem surpresa em produção.

5

Distribuir

Instalador .exe pronto ou migração pra C# nativo.

Como instalar

Instalação em 1 clique

A aula entrega o instalador pronto. Sem terminal, sem comandos, sem configuração.

1

Acesse a aula Claude Code Skills

No curso, baixe o arquivo .exe de instalação que está anexado.

2

Execute o instalador

Dois cliques no .exe. O wizard cuida de tudo: verifica Git, Node.js e Claude Code CLI, baixa o plugin do GitHub e registra automaticamente.

3

Reinicie o Claude e digite '/'

No VS Code, reinicie a sessão do Claude Code e digite / pra ver todas as skills disponíveis.
